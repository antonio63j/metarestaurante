Actualiza las imagenes de página principal

Para actualizar las imagenes, deberemos registrarnos e iniciar sesión con perfil de administrador.
Podremos tener hasta tres imagnes que irán deslizandose por la página inicial. Deberemos tener en cueta que todas deberán tener las mismas demensiones y no ser demasiado pesadas pera no demorar la carga de las páginas. Para conseguirlo, podemos tratar las imágenes con algún software específico para ello.
